    Markdown extensions

    Beyond the basics, Docodile supports more advanced Markdown syntax. Many you will have seen in other Markdown processors, like Admonitions and Tables which are supported by most major static site generators.

    Some extensions in Docodile are not necessarily available in other Markdown processors and therefore might not work in both.
